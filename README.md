## Chat API

Chat API is a Node.js backend server written in TypeScript.

## Installation

Clone the repository

```bash
$ git clone https://github.com/bwen19/chat-api.git
```

Switch to the repo folder

```bash
$ cd chat-api
```

Install dependencies

```bash
$ npm install
```

Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
